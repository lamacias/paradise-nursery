A front-end shopping application for a houseplant nursery, built with React.

## Live Demo

[View the live application](https://YOUR-USERNAME.github.io/paradise-nursery/)

## Features

### Landing Page
- Beautiful background image with gradient overlay
- Company name and description
- "Get Started" button to navigate to products

### Product Listing Page
- Header with shopping cart icon showing item count
- 6 houseplants organized into 3 categories:
  - Tropical Paradise
  - Low Light Lovers
  - Succulent Collection
- Each plant displays: thumbnail, name, description, price, and "Add to Cart" button
- Dynamic cart count updates

### Shopping Cart Page
- Displays total number of plants and total cost
- Each item shows: thumbnail, name, unit price, quantity controls, and delete button
- Increase/decrease quantity buttons
- "Continue Shopping" and "Checkout" buttons

## Technologies Used

- React 18
- Tailwind CSS
- Vanilla JavaScript

## How to Run Locally

Simply open `index.html` in a web browser.
